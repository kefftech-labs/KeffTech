# 👑 KeffTeckLabs - Hub Suprême PWA

Bienvenue dans la matrice **KeffTeckLabs**, l'interface Multi-PWA (Progressive Web App) conçue par l'Architecte **Shemoel Hai**. Ce projet n'est pas qu'un site web, c'est un "condensateur de Vent Blanc", un Hub centralisant plusieurs réalités technologiques et sociales.

## 🌀 Vision & Philosophie (Le ROI Subatomique)
Nous transformons le "Vent Blanc" en "Or physique". Ce Hub est l'interface numérique de projets matériels et IA (GoleMotor, Titan-Rack, Jewel-Phone). 
Investir ici ou s'abonner, c'est participer au **Tikoun Olam matériel** (réparer le monde par l'action concrète).

## 🚀 Fonctionnalités Principales (L'Architecture 1 MW)
Ce Hub est construit comme une **Single Page Application (SPA)** fluide, utilisant un "Skeleton Glass Design" avec des effets holographiques animés.

* **📱 Multi-Plateformes intégrées :**
    * **Hub (Home) :** Le Manifeste ROI et les abonnements Multipass.
    * **Bonusino :** Le moteur de troc et d'échange (Connecté à Supabase).
    * **LibrarchiA :** L'interface SaaS pour l'IA Emet.
    * **ViA Core OS :** Le panneau de contrôle du système.
    * **GoleMotor :** Le suivi de l'établi physique (Titan-Rack, Smartwear).
* **🌈 Text-Rainbow Engine :** Une classe CSS (`.text-rainbow`) qui anime les textes clés avec le prisme complet de la création.
* **🧭 Navigation Magique :** Scrolling vertical naturel (anti-friction) avec menu inférieur et Sidebar de gestion de compte.
* **🎵 Feedback Haptique Sonore :** Chaque interaction génère un "Keff Sound" (Note C5 à C6) pour confirmer l'action.

## 🛠️ Stack Technique
* **Frontend :** HTML5, CSS3 (Glassmorphism, Animations), Vanilla JavaScript.
* **Backend / BaaS :** Supabase (via CDN).
* **Architecture :** Zéro framework lourd. "Zéro Fiction". Code natif pour une vitesse d'exécution maximale.

## 📥 Installation & Utilisation
Le projet est prêt à être utilisé directement dans le navigateur (ou compilé via Capacitor/Cordova pour les stores).

1. Clonez ce dépôt.
2. Ouvrez le fichier `index.html` dans un navigateur moderne.
3. Configurez vos clés Supabase dans le script JS pour activer le module Bonusino.
